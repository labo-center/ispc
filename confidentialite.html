Politique de confidentialité

Dernière mise à jour : février 2026

En résumé : vos données ne quittent jamais votre appareil. PRÉVIO fonctionne entièrement en local (localStorage du navigateur). Aucune donnée personnelle n'est collectée, transmise ou stockée sur un serveur externe.

1. Architecture locale (local-first)

PRÉVIO est conçu selon une architecture « local-first ». Cela signifie que toutes les données que vous saisissez (réponses aux questionnaires, scores, profil) sont stockées exclusivement dans le localStorage de votre navigateur web.

Concrètement :

  • Aucune base de données distante n'est utilisée
  • Aucun compte serveur n'est créé
  • Aucune API de données personnelles n'est appelée
  • Le code s'exécute intégralement côté client (dans votre navigateur)

2. Données stockées

Les données enregistrées dans votre navigateur incluent :

  • Profil local : prénom ou pseudo (aucun email, aucun mot de passe)
  • Réponses aux questionnaires : données contextuelles (âge, taille, poids, habitudes), réponses aux modules santé
  • Scores calculés : HOMA-IR, ISI, GAD-7, DASH, etc. — calculés localement à partir de vos réponses
  • État de progression : quels modules ont été complétés
  • Checklist : état du plan d'action (coché/non coché)

3. Données NON collectées

PRÉVIO ne collecte pas :

  • Adresse email
  • Mot de passe
  • Adresse IP (pas de tracking serveur)
  • Données de géolocalisation
  • Cookies de tracking ou publicitaires

4. Hébergement et services tiers

Le site est hébergé sur Vercel. Les seules requêtes réseau sont :

  • Le chargement des pages HTML/CSS/JS (fichiers statiques)
  • Le chargement des polices Google Fonts (DM Sans, Fraunces)

Vercel peut collecter des logs d'accès (adresse IP, user-agent) dans le cadre de son service d'hébergement. Consultez la politique de confidentialité de Vercel pour plus de détails.

Google Fonts peut collecter des données dans le cadre du chargement des polices. Consultez la politique de confidentialité de Google.

5. Export et suppression de vos données

Vous avez un contrôle total sur vos données :

  • Export : vous pouvez exporter toutes vos données au format JSON depuis la section « Données locales » de l'application
  • Import : vous pouvez restaurer vos données depuis un fichier JSON exporté précédemment
  • Suppression : vous pouvez supprimer toutes vos données à tout moment depuis la section « Données locales ». Cette action est irréversible.
  • Effacement navigateur : vider le cache ou le localStorage de votre navigateur supprimera automatiquement toutes les données PRÉVIO

6. Sécurité

Mesures de sécurité mises en place :

  • Content Security Policy (CSP) restrictive
  • Protection contre l'injection XSS (sanitization des entrées)
  • Validation stricte à l'import JSON (schéma, types, taille)
  • Aucune donnée sensible en clair (pas de mot de passe stocké)
  • Headers de sécurité : X-Content-Type-Options, X-Frame-Options, Referrer-Policy

7. Mineurs

PRÉVIO n'est pas destiné aux personnes de moins de 16 ans. L'utilisation par un mineur doit se faire sous la supervision d'un parent ou tuteur légal.

8. Modifications

Cette politique peut être mise à jour. La date de dernière modification est indiquée en haut de cette page.
